Modus

Modus = nilai yang paling sering muncul

atau

Modus = nilai yang frekuensinya paling tinggi

Contoh Soal

1 2 3 4 5

Soal :
Tentukan modus dari data berikut ini.

4, 2, 3, 6, 7, 3, 5, 3, 2, 5, 3

Pembahasan :
Untuk mengetahui modus dari data tersebut, isilah kolom di bawah ini:

  • banyak nilai 2 :
  • banyak nilai 3 :
  • banyak nilai 4 :
  • banyak nilai 5 :
  • banyak nilai 6 :
  • banyak nilai 7 :

Nilai yang paling sering muncul adalah 3, yaitu sebanyak empat kali. Jadi, modus data tersebut adalah 3.

Soal :
Tentukan modus dari data berikut ini.

8, 10, 9, 10, 8, 6, 7, 9, 10, 9, 7

Pembahasan :
Untuk mengetahui modus dari data tersebut, isilah kolom di bawah ini.

  • banyak nilai 6 : 
  • banyak nilai 7 : 
  • banyak nilai 8 : 
  • banyak nilai 9 : 
  • banyak nilai 10 :

Nilai yang paling sering muncul adalah 9 dan 10, yaitu sebanyak tiga kali. Jadi modus data tersebut adalah 9 dan 10. Karena modusnya ada dua, maka disebut bimodus.

Soal :
Tentukan modus dari data berikut ini.

4, 5, 6, 8, 9, 10, 7

Pembahasan :
Untuk mengetahui modus dari data tersebut, isilah kolom di bawah ini:

  • banyak nilai 4 : 
  • banyak nilai 5 : 
  • banyak nilai 6 : 
  • banyak nilai 7 : 
  • banyak nilai 8 : 
  • banyak nilai 9 : 
  • banyak nilai 10 :

Karena, setiap nilai pada data hanya muncul satu kali, maka tidak ada modus dari data tersebut.
